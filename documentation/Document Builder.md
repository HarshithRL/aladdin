# **Document Builder Module**

The **Document Builder Module** is an innovative feature of Etex Mate, designed to simplify the process of creating, editing, and managing files of various formats. Whether you're building markdown documents, programming scripts, or plain text files, this module empowers users to craft and customize their content with ease.

---

## **Key Use Cases**  
1. **File Creation by Prompt**:  
   - Users can create files by providing a descriptive prompt directly in the chat.  
   - The module supports a wide range of file types, including markdown, plain text, blogs, programming scripts (Python, HTML, JavaScript, etc.), and more.  

2. **File Editing Tools**:  
   - Upload an existing file or paste its content into the editor to apply modifications.  
   - Modify, undo, redo, and save changes effortlessly.  

3. **Formatted Previews**:  
   - Preview files like markdown, plain text, or blog content in a clean, formatted style.  
   - Note: Preview isn’t available for code files such as Python, HTML, or JavaScript.

4. **Download and Share**:  
   - Download created or edited files in their respective formats.  
   - Copy file content directly from the editor for immediate use.

---

## **Capabilities of the Document Builder Module**

### **1. Comprehensive File Creation**  
Simply instruct Etex Mate via chat to create a file in your desired format and include your content idea.  
- Examples:  
  - "Create a markdown file for meeting agenda."  
  - "Generate Python code for a calculator program."  
  - "Write a blog about sustainable architecture."  

The system will prepare the file, ready for download or further customization.

---

### **2. Flexible Editing Options**  
Users can edit files interactively through a full-featured editor.  
Key Features Include:  
- **Edit and Save**: Make changes to a file directly in the editor and save modifications by clicking the save button. Note: Changes aren’t saved unless explicitly saved by the user.  
- **Undo and Redo**: Navigate seamlessly through your edits without fear of losing progress.  
- **Content Replacement**: Paste any content into the editor and either continue work or ask Etex Mate to create a new file in the same format.

---

### **3. Preview and Export**  
Upon file creation or upload:  
- **Preview Capabilities**:  
  - Markdown, plain text, and blog files can be previewed in a clean, formatted layout.  
  - Code files (e.g., Python, HTML, JavaScript) cannot be previewed but remain available for editing and download.  
- **Export Features**:  
  - Download files in their respective formats—ready to share or deploy.  
  - Copy file content for quick sharing or further use elsewhere.

---

## **How It Works**  

1. **Creating a File via Chat Prompt**  
   - Tell Etex Mate what type of file you want and provide the content idea:  
     Example: "Generate a JavaScript function for form validation."  
   - Etex Mate will create the file and save it in the specified format.

2. **Editing Files**  
   - Paste the content of your current file into the editor or upload the file.  
   - Modify the content and utilize undo/redo functionality as needed.  
   - Save changes by clicking the save button to ensure the latest version is stored.

3. **Preview and Export**  
   - If the file is markdown, plain text, or blog format, preview it in its final, formatted view.  
   - If it’s a code file, skip to editing or download it directly.  
   - Download the file once finalized or copy its entire content in one step.

4. **Apply Changes via Chat**  
   - Ask Etex Mate to make updates or modifications to the current file.
   - The updated file will always be provided in full, ensuring users have the complete and most recent version.
