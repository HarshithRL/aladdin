# **Etex Mate: Comprehensive Guide**

Etex Mate is an advanced AI-driven productivity and collaboration tool designed to streamline your workflow and enhance your decision-making with intelligent capabilities. This document serves as a comprehensive guide to its features, capabilities, and agent architecture.

![Etex Chat Overview](/static/documentation/3.png)

---

## **Introduction to Etex Mate**

Etex Mate enables users to:
- Interact intelligently with AI to complete tasks, answer queries, and generate content.
- Collaborate seamlessly with colleagues through tagging and file sharing.
- Leverage multi-agent architecture for specialized functionalities like image generation, document analysis, and Microsoft 365 integrations.
- Access structured and visual outputs, including charts, tables, and detailed summaries.

## **Agents**

Etex Mate consists of a suite of specialized agents. Each agent is purpose-built to handle specific tasks, making Etex Mate highly versatile.

---

### **1. Etex Chat Agent**
![Etex Chat Logo](/static/documentation/3.png)

Primary capabilities:
- Answering diverse queries across technical, creative, and general domains.
- Generating text, code, and structured documents.

---

### **2. Image Generator Agent**
![Image Generator Agent Logo](/static/documentation/30.png)

Specializes in creating images based on descriptive prompts. Built for:
- Visual mockups
- Graphic design assistance
- Marketing assets

---

### **3. Document Analyst Agent**
![Document Analyst Agent Logo](/static/documentation/31.png)

Processes and analyzes uploaded documents:
- Supported formats: `.pdf`, `.doc`, `.docx`, `.xls`, `.xlsx`, `.csv`, `.ppt`, `.pptx`.
- Core capabilities: Summarization, chart generation, and data analysis.

---

### **4. Etex Microsoft 365 Search Agent**
![Etex Microsoft 365 Agent Logo](/static/documentation/29.png)

Fully integrates with Microsoft 365 to enable:
- Advanced search across SharePoint, OneDrive, Outlook, and Teams.
- Calendar management, including retrieving upcoming meetings.
- Email drafting, editing, and direct sending.
