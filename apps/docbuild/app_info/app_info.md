# Document Intelligence

## Your AI-Powered Document Analysis Assistant

Welcome to Document Intelligence, an advanced document analysis application within the Etex Mate web platform designed exclusively for Etex employees. This powerful tool allows you to upload documents and engage in natural conversations about their content, extracting insights and information with remarkable accuracy.

### Key Features

- **Document Analysis**: Upload files and create interactive sessions where you can ask questions about document content.

- **Natural Language Interaction**: Ask questions in plain language and receive clear, contextual responses based on your documents.

- **Model Selection**: Choose from various AI language models using the dropdown menu in the top left corner to best suit your specific document analysis needs.

- **Intuitive Interface**: Each response includes convenient buttons to copy text and provide feedback with like/dislike options.

- **Session Management**: 
  - Access all previous document analysis sessions from the sidebar
  - Create and customize session groups for better organization
  - Rename or delete sessions as needed
  - Change group colors for visual organization
  - Rename or delete groups

### Data Security Commitment

Your privacy and data security are our highest priorities at EtexChat:

- All data remains within Etex's Azure subscription
- Your prompts, documents, transcripts are:
  - NOT available to other customers of Open AI
  - NOT available to OpenAI
  - NOT used to improve OpenAI models
  - NOT used to train, retrain, or improve Azure OpenAI Service foundation models
  - NOT used to improve any Microsoft or 3rd party products/services without your permission

**For more details, read Microsoft's official privacy policy** [here](https://learn.microsoft.com/en-us/legal/cognitive-services/openai/data-privacy?tabs=azure-portal).  

### How Document Intelligence Works

Document Intelligence utilizes Retrieval-Augmented Generation (RAG) technology with fine-tuned Azure OpenAI models that are exclusively available for Etex use. The service operates entirely within Microsoft's Azure environment and does not interact with any services operated by OpenAI (such as ChatGPT or the OpenAI API).

### User Experience

Document Intelligence is designed to provide:
- Clear, concise answers to your document-related questions
