# What's New in Etex Mate Chat - Latest Update

We’re excited to announce a major update focused on powerful new features, performance improvements, and critical bug fixes to make your experience smoother than ever.

## New Features
- Search seamlessly across Outlook, SharePoint, OneDrive, and Teams — all in one unified query.

- Quickly upload and share documents directly within conversations.

- Share screenshots, charts, and images inside your chats with ease.

- Helpful tooltips assist you in choosing the right Language Model for your task.

- Chat responses now adapt intelligently based on your user profile and session context.

- Effortlessly copy tables to your clipboard or download them for external use.

- Switch to a comfortable, sleek dark theme to reduce eye strain.

- A consistent welcome message is shown when you open the app.

## Improvements
- File uploads, previews, and processing speeds have been significantly improved.

- The app now gives clear, actionable messages when it cannot complete a request.

- Replies follow a cleaner, more structured format for easier reading and interpretation.

## Bug Fixes
- Fixed an issue where the copy button disappeared from code snippets.

- Code blocks now correctly display syntax highlighting and background colors.

- Minor inconsistencies across the interface have been addressed for a more polished look.

- Stability improvements and fixes related to using template pop-ups.

- Resolved problems that affected saving or editing newly created templates.

- Fixed an issue where session group labels showed incorrect colors.
