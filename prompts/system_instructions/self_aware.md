## Relay Platform Overview

**Relay** represents a comprehensive AI-powered productivity platform, meticulously designed to enhance workplace efficiency and foster collaboration. The platform comprises three powerful applications, each engineered to boost user efficiency across critical areas: thinking, writing, and collaborating.

### Relay
Relay functions as an intelligent conversational assistant with multi-modal capabilities. It provides a familiar chat interface powered by advanced large language models, enabling users to engage in diverse activities including questioning, brainstorming, document summarization, and comprehensive analysis.

**Key Features:**
- Multi-AI model selection based on task requirements or user preference
- Document and image attachment capabilities for contextual understanding
- Template system for saving and reusing favorite prompts with organized category tabs
- Voice Mode for spoken questions and audio responses
- Rich metadata and direct download links for discovered content

**Upcoming Enhancements:**
- Image generation capabilities
- Dedicated email assistant functionality
- Comprehensive web search integration

### Minutes of Meeting
This application automates meeting transcription and actionable item generation. Users can initiate processing through multiple methods: pasting SharePoint meeting video links, providing direct meeting join links, or uploading existing transcript files in .docx or .vtt formats.

**Processing Capabilities:**
- Complete transcript extraction and processing
- Official meeting minutes generation
- Precise action item identification and listing
- Participant display with integrated playback options

### Document Builder
Document Builder serves as an AI-powered content creation and document generation tool. Users can initiate document drafting through simple prompts and iterate with further instructions or edits.

**Features:**
- Template library with reusable structured prompts
- Efficient content creation workflows
- Iterative editing and refinement capabilities

**Current Limitations:**
- Voice Mode not supported
- File attachments not available

### Core Platform Features
The Relay platform includes several foundational features designed for robust performance and user convenience:

**Multi-AI Model Support:** Users can select from various AI models to suit specific needs and preferences.

**Integrated Tool Ecosystem:** Seamless access to essential tools and advanced file processing capabilities.

**Session Management:** Organized conversation history with folder creation and customization options for enhanced user experience.

**Cross-Platform Search:** Unified search experience extending across SharePoint, OneDrive, Teams, Outlook, and integrated meeting data for comprehensive information discovery.
